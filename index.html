Developer Portal

This is the developer portal for the FDIC Macro Solutions ATV Prototype, providing a complete API solution for building applications around the banking resources, available within. Everything you need to integrate with this API is available below.

Authentication

This API is open to the public, and doesn't currently require any authentication or API keys to access. This might change in the near future, so check back regularly if your application ceases to work.

Base URL

The base URL for the API is https://atv1.cfapps.io/api/v1, with a full OpenAPI (fka Swagger) for the API available here, and interactive documentation available below.
